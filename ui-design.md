# UI Design

## Status

Unvibe Code has no graphical product UI. Its interaction surface is the agent
skill conversation and generated learning material.

## Interaction Design

- Teach interactively instead of dumping a full codebase explanation.
- Use small checkpoints that test understanding before moving deeper.
- Keep terminology tied to the inspected source.
- Separate observed behavior from inference.
- Prefer concise diagrams or examples when they materially improve learning.

If a graphical interface is added, document its implemented flows,
accessibility rules, tokens, and source paths here.
